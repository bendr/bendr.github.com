Under Development

Bender is an experimental, declarative Web application authoring framework based on XML, Javascript and Web technologies, and with a focus on rich graphical user interfaces.

Download Bender v0.8.1, released 17 June 2013. Bender is free software and is released under the Apache License v2.0. The source code is hosted on Github.
